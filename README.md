# CommunityNode

## Description

A node running by the community, keeping the private key with encryption, executing all instructions send by gateway.

Get a quick way, use Etherspot SDK!!!

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
## create batch
$ pnpm execute batch-create {count}

ex
```
pnpm execute batch-create 2
```
